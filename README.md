# yStnadard Blocks

無料WordPressテーマ「yStandard」用Gutenbergブロック追加プラグイン

「yStandard」用Gutenbergブロック追加・拡張プラグイン

### 追加ブロック

* ボタン型リンク
  * HTML入力欄にアンカー(a)タグを貼り付けるとボタン型に表示するブロック
  
### 標準ブロック拡張

* インライン装飾
  * 3種類のインライン装飾ボタンを追加。カスタマイザーから装飾内容をカスタマイズ可能
* ボタンブロック拡張
  * コアのボタンブロックにアイコン表示、ブロック型表示ができるオプションを追加
